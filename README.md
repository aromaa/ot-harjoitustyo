# P2P Chat

Sovelluksen avulla käyttäjät voivat jutella ja mahdollisesti jakaa tiedostoja P2P yhteyden avulla. Käyttäjä voi vastaanottaa ja muodostaa useampia viestejä samanaikaisesti erihenkilöiötä.
